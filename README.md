helpers - сторонние методы
modules - разделены на tabs и страницы. Реализация через POM и Page/TabFactory
testData - тестовая дата: картинки, объекты
tests - тесты + хелсчек
utils - настройка тестов, фикстуры и глобал сетап

Тесты:
1 Health check:
- открытие стартовой страницы
- проверка соответствия урла

2. Open ${название табы} tab and assert its content (параметризованный тест):
- открытие стартовой страницы
- клик и открытие целевой табы
- ожидание прогрузки элементов
- проверка, что модуль с элементами виден

3. Assert sending face detection API-request (АПИтест):
- Открытие страницы
- формирование и отправка запроса через fetch, получение ответа
- проверка, что полученный ответ содержит все поля конфигов, которые были в запросе
